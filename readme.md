# CODERHOUSE EXCHANGE

Proyecto de pre-entrega Nº1 para el curso de Javascript de coderhouse.

## Descipción

Simulador que permite la conversion de montos entre distintas divisas, utilizando la tasa de cambio fijada en el aplicativo.<br>
El usuario puede seleccionar moneda de origen y destino y repetir operaciones hasta cancelar.

## Funcionalidades

- Ingreso monto con validación.
- Selección de moneda origen y destino.
- Conversion automática.
- Manejo de cancelación.
- Posibilidad de realizar multiples operaciones.

## Tecnologias
- Javascript
- HTML
- CSS

## Ejecución
1. Descargar el repositorio.
2. Abrir el archivo 'index.html' en el navegador.
3. Ingresar monto y moneda.
4. Visualizar conversión.
